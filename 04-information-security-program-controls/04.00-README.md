## What this phase is

This phase builds the control program that satisfies GLBA §501(b) and **treats the 8 High risks** from the risk assessment. Its keystone is the **Written Information Security Program (WISP)** — the written, board-approved program the Interagency Guidelines require — supported by **14 core policies** and organized into administrative, technical, and physical safeguards. Every control is traced back to the risk it treats and forward to a NIST CSF 2.0 function, so the program is provably complete and examiner-ready.
