## Context
GLBA §501(b) and the Interagency Guidelines require a **written**, board-approved information security program commensurate with the risk assessment.

## Decision
Adopt a **Written Information Security Program (WISP)** structured around administrative, technical, and physical safeguards, supported by 14 core policies, mapped to the 8 High risks and NIST CSF 2.0.

## Alternatives Considered
**Rely on scattered procedures** — rejected: not a defensible written program. **Framework-only, no policies** — rejected: not operational.

## Consequences
A single, board-approved program that satisfies §501(b) and organizes all controls.
