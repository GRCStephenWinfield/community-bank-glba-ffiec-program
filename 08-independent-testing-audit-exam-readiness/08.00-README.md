## What this phase is

Everything the program claims now gets tested by someone independent. This phase runs the **GLBA-required independent testing** — an external **penetration test by Redwood Security Partners** — plus the **internal audit** of the program, then prepares for and undergoes the **FFIEC IT examination** and supports the **SOX 404(b) external audit**. Five independent lenses look at the same program, and it holds up: the pen test findings are all remediated, internal audit is Satisfactory, the IT exam lands **Satisfactory (URSIT composite "2")**, and the external auditor issues an unqualified opinion on ICFR.
