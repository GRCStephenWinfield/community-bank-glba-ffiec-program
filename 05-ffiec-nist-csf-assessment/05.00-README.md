## What this phase is

This is Cornerstone's **cybersecurity maturity assessment** — the FFIEC cybersecurity self-assessment, conducted through **NIST CSF 2.0** because the FFIEC Cybersecurity Assessment Tool (CAT) was retired on 2025-08-31. It scores the program's current maturity across all **six CSF 2.0 functions** (Govern, Identify, Protect, Detect, Respond, Recover — 22 categories, 106 subcategories), sets a **target profile of Intermediate** (right-sized to the Bank's Moderate inherent risk), and produces the consolidated **28-gap register** and a time-phased remediation roadmap. It shows the program's shape honestly: Protect is strongest (the Phase 04 controls), while Detect, Respond, and Recover need the most maturity work.
