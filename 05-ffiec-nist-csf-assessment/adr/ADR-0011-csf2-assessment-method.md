## Context
The FFIEC Cybersecurity Assessment Tool (CAT) was sunset 2025-08-31. Cornerstone needs a defensible cyber maturity assessment method examiners recognize.

## Decision
Conduct the FFIEC cybersecurity assessment using **NIST CSF 2.0** (6 functions, 22 categories), retaining the CAT inherent-risk-profile structure and mapping current→target profiles; reference the CRI Profile as an alternative crosswalk.

## Alternatives Considered
**Keep using the retired CAT** — rejected: unsupported. **CRI Profile only** — viable but CSF 2.0 is the broadest, best-mapped standard.

## Consequences
A current, examiner-appropriate assessment with a clean crosswalk and roadmap.
