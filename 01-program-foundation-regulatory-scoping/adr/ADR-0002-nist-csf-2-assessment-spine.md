## Context
The FFIEC Cybersecurity Assessment Tool (CAT) was officially sunset on 2025-08-31. Cornerstone needs a forward-looking, examiner-recognized assessment method.

## Decision
Adopt **NIST CSF 2.0** (Govern/Identify/Protect/Detect/Respond/Recover) as the cybersecurity assessment spine, retaining the CAT's inherent-risk-profile structure and mapping current→target profiles. Reference the CRI Profile as an alternative crosswalk.

## Alternatives Considered
**Continue using the retired CAT** — rejected: no longer supported. **CRI Profile only** — viable but CSF 2.0 is the broadest common denominator and maps cleanly to FFIEC expectations.

## Consequences
Current, defensible assessment; smooth crosswalk to FFIEC booklets and SOX; forward maturity roadmap.
